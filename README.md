# Article Studio

Upload any media except video to generate a social-media-ready post.

## Workflow

1. Upload an article or media file.
2. The app extracts text from the file.
3. It generates a title first.
4. Accept or revise the title until you are happy.
5. The app generates a summary after the title is accepted.
6. Accept or revise the summary.
7. Generate a social media blurb for your chosen platform.
8. Optionally generate an image for the article.

## Notes

- Supported uploads include markdown, text, PDF, RTF, DOCX, DOC, and common image formats.
- Video files are not supported.
- Set your `OPENAI_API_KEY` in a local `.env` file before running the app.
